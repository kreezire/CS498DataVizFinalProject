About this Visualization

This visualization uses the Interactive Slideshow to tell the story of Regions of the World. It tells some stats about primary geographical regions and then dives into some details for regions in Europe.
Visualizations are built with various annotations, parameters and some triggers.

Elements
  1. Scenes

    2. There are multiple slides, including a few charts in this presentation.

  2. Navigation

    3. Each slide can be navigated through guiding arrows in the right bottom corner. At any point, press ESC key to see and overview of all slides.

  3. Parameters

    4. Main parameter of each chart is "region". Each chart is separate and deals with a few other parameters like net migration, GDB, Literacy rate and contribution of various economic sectors.

  4. Triggers

    5. Some charts have triggers. Like: Zoom in scatter plot.

  5. Annotations

    6. Charts include legends, tool tips and labels for hinting.

The Database
All these data sets are made up of data from the US government. Generally they are free to use if you use the data in the US. If you are outside of the US, you may need to contact the US Govt to ask. Net Migration vs GDP (scroll down) This chart shows how Net Migration is varies with GDP. Each mark on chart represents a country. Each dot is color codded to represent its region.
You can select region from legend to see details of a specific region.
You can zoom along x and y axis. Use "Reset Zoom" button to restore the view.
Tool-tip is available to check values for any plotted point. A semi-transparent tool tip box will appear when you hover mouse pointer over the point.

It can be noted from the chart that for countries with lower GDP net migration is mostly negative- suggesting that people out of these countries.
And for countries with higher GDP net migration is mostly positive- suggesting that more people into these countries.
